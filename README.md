# FPGA_Pong_Game

Designed and implemented a two-player Pong game in VHDL on an FPGA using an 8-LED array to display ball movement and seven-segment displays to track player scores. 

The project was built using a modular structural design that integrated a finite state machine controller, clock divider, score counter, input buffer, LED display driver, seven-segment decoder, multiplexers, and a top-level integration module. The game logic handled ball movement, player inputs, collision detection, score updates, and selectable game speeds through synchronized hardware components. This project strengthened my experience with FPGA development, structural VHDL design, finite state machines, clock management, and real-time digital system implementation.
